This is a repo containing patches and scripts to get a cross compiler from amd64 linux to powerpc mac os x working.
See build.sh and the tests folder for more info.

Projects used:
cctools-port (https://github.com/tpoechtrager/cctools-port)
apple's version of gcc (see their opensource website)
a legacy version of bison (http://gnuwin32.sourceforge.net/packages/bison.htm)

Dependencies: build-essential, git, wine, ssh
